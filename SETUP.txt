Install Dependancies Command:
sudo apt update && sudo apt install -y cmake libopencv-dev python3-opencv libserial-dev libraspberrypi-dev pkg-config build-essential

Build Command (from /RobotFramework directory):
rm -rf build && mkdir build && cd build && cmake .. && make
